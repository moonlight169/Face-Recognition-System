=========================================
Face Recognition System (Auto-Folder & Smart % Version)
=========================================

ระบบตรวจจับและจดจำใบหน้าด้วย OpenCV (Python)
อัปเกรดใหม่: จัดการรูปภาพง่ายขึ้นด้วยระบบแยกโฟลเดอร์ และแสดงผล % ความเหมือนที่แม่นยำขึ้น

[ โครงสร้างไฟล์และโฟลเดอร์ ]
โปรเจกต์ของคุณควรมีหน้าตาแบบนี้:
📁 Face_Recognition_Project/
 ├── 📄 main.py (โค้ดหลักของโปรแกรม)
 ├── 📄 haarcascade_frontalface_default.xml (ไฟล์ AI ตรวจจับใบหน้า)
 ├── 📄 trainer.yml (ไฟล์สมองกล ระบบจะสร้างให้เอง)
 └── 📁 dataset/ (โฟลเดอร์หลักสำหรับเก็บรูปภาพ)
      ├── 📁 Id1/ (โฟลเดอร์ของคนที่ 1 เช่น Light)
      │    ├── รูปอะไรก็ได้.jpg
      │    └── IMG_1234.png
      └── 📁 Id2/ (โฟลเดอร์ของคนที่ 2)
           ├── pic_1.jpg
           └── image.jpg

[ วิธีเตรียมรูปภาพ (Dataset) ]
1. สร้างโฟลเดอร์ชื่อ dataset
2. ข้างในโฟลเดอร์ dataset ให้สร้างโฟลเดอร์ย่อยตั้งชื่อตาม ID เช่น Id1, Id2, Id3
3. เอารูปหน้าตรงของแต่ละคน โยนใส่เข้าไปในโฟลเดอร์ของคนนั้นๆ ได้เลย **(ไม่ต้องมัวมานั่งเปลี่ยนชื่อไฟล์แล้ว! ระบบจะฉลาดพอที่จะดึงข้อมูลจากชื่อโฟลเดอร์แทน)**
4. แนะนำคนละประมาณ 30-50 รูป (ยิ่งมีมุมกล้อง สภาพแสง หรือใส่แว่น/ถอดแว่นหลากหลาย AI ยิ่งฉลาด)

[ วิธีตั้งชื่อแสดงผลบนหน้าจอ ]
เข้าไปแก้ในไฟล์ main.py ตรงบรรทัดที่เขียนว่า:
names = ['None', 'Light'] 
(Id1 จะไปดึงชื่อจากตำแหน่งที่ 2 ในลิสต์ ถ้ามี Id2 ก็พิมพ์ชื่อต่อท้ายเข้าไปได้เลย เช่น ['None', 'Light', 'Kira'])

[ วิธีรันโปรแกรม ]
1. เปิด Terminal ใน VS Code หรือ CMD
2. พิมพ์คำสั่ง: python main.py
3. การทำงาน:
   - รันครั้งแรก (หรือตอนไม่มีไฟล์ trainer.yml): ระบบจะวิ่งเข้าไปอ่านโฟลเดอร์ Id ต่างๆ เพื่อเทรนข้อมูล แล้วเปิดกล้อง
   - รันครั้งต่อไป: ระบบจะข้ามการเทรน โหลดข้อมูลเดิมมาใช้ แล้วเปิดกล้องทันที 
4. การปิดโปรแกรม: คลิกที่หน้าต่างกล้อง แล้วกดปุ่ม 'q' บนคีย์บอร์ด

[ วิธีสอนให้จำคนใหม่ หรืออัปเดตรูปใหม่ ]
ถ้าเอารูปใส่เพิ่มในโฟลเดอร์ หรือเพิ่มโฟลเดอร์ Id ใหม่เข้าไป
-> ให้ทำการ "ลบไฟล์ trainer.yml ทิ้ง" ซะ
-> แล้วรันคำสั่ง python main.py ใหม่อีกรอบ ระบบจะกวาดรูปสร้างสมองกลขึ้นมาอัปเดตให้เอง!
